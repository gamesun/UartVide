Version history (What's new?)

2.5 (2022-03-07)

  • Optimize the process of receiving.
  • UI fine tuning.
  • Add: resize the form by dragging window borders.
  • Add: Rx Tx counters.
  • Add: rename QuickSend button dailog(by right-click).
  • Add: select file buttons in QuickSend table.
  • Add: timestamp, send mode, loop time status save in settings file.
  • Add: detect busy state of ports
  • Fix: sometimes open port button keep active when port closed.
  • Fix: lost output when decode exception happened in ASCII mode.
  • Fix: port select combobox become empty after refreshing

2.4 (2021-06-06)

  • MyTerm was renamed to UartVide
  • New program icon.
  • Use Balloon-Tip instead of Messagebox.
  • Use PySide2 instead of PyQt5.
  • UI fine tuning.

2.3.0 (2020-08-20)

  • Rework PortConfig panel.
  • Changing port config don't need restart port.
  • Display port friendly name in combobox.
  • Add Periodic Send.
  • Set tab size = 4.

2.2.0 (2020-05-07)

  • A cleaner user interface and a new icon.
  • Messagebox, open/save dialog use default style instead of flat style.
  • Open/save dialog show at screen center now.
  • Add QuickSend options menu.
  • Auto-save QuickSend table contents.
  • Auto close port when serial cable is plugged out.
  • Installer no longer requests administrator permissions.
  • Move settings folder to install folder.
  • Fix: refreshing ports causes add repeating items in combobox.
  • Fix: refreshing ports causes port select changed.
  • Fix: "RuntimeError: wrapped C/C++ object of type xxx has been deleted" at exit.
  • Fix: timestamp loss '.000' when microsecond is 0.
  • Fix: titlebar disappears at linux.
  • Fix: view mode dosen't work.

2.1.0 (2018-11-06)

  • Read all data from receive buffer at a time.
  • Fix: hex(lowercase),HEX(UPPERCASE) view mode dose not work.
